Adds a PLOS Search Widget to your site. Users can use it to search for PLOS articles.

== Description ==

[PLOS](http://plos.org) is a non-profit open access journal publisher and advocacy organization. This plugin adds a search widget to your site that lets users search the through the PLOS corpus of academic, peer-reviewed science articles.

== Changelog ==

= 0.1 =
* Disable SSL for now
* Based on [DuckDuckGo Search Widget](http://wordpress.org/plugins/duckduckgo-search/)

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for the PLOS Search Widget under `Appearance -> Themes`, and add it to your layout
4. Your users can now use the newly created search box to search for PLOS
articles
